# Azure Centralized Logging & Log Retention Management

## Overview
This project demonstrates centralized logging, monitoring, retention governance, and alerting using Azure-native services.

## Architecture
Azure VMs + Azure SQL
        ↓
Azure Diagnostic Settings
        ↓
Log Analytics Workspace
        ↓
KQL Queries + Azure Workbooks
        ↓
Azure Monitor Alerts → Action Groups

## Features
- Centralized log aggregation
- KQL-based log analysis
- Real-time alerting
- Retention optimization
- Incident-based log investigation
- Governance and compliance awareness
